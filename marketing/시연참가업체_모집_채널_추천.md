## ★★★ 1순위 — 기존 인맥 / 레퍼런스 / 제휴

가장 적은 노력으로 **1호 시연사**를 확보하는 경로. 모집의 시작점.

- **사내·대표 인맥**: 기존 거래 부품사, 전 직장 동료가 있는 품질팀
- **품질 컨설팅 / IATF 인증 심사기관 제휴**: 이들의 고객사가 곧 타겟. 수수료·공동제안 구조
- **설비·MES·ERP 벤더 제휴**: 부품사 IT 도입 담당과 연결
- **기존 FMEA Excel 500개 출처사**: 데이터를 보유한 협력사 네트워크부터 우선 접촉

## ★★ 2순위 — 품질 실무자 온라인 커뮤니티

표준류를 **실제로 작성하는 실무자**에게 직접 닿는 저비용 채널.

### 네이버 카페 (실무자 다수 활동)
- 품질경영시스템 / ISO·IATF 인증 실무 카페
- 자동차 부품 품질·생산기술 실무 카페
- APQP·FMEA·Control Plan 등 핵심 키워드로 카페 검색 후, **운영 활발·실무자 비중 높은 곳** 2~3개 선정
- 게시 방식: 정보성 글(예: "FMEA 작성시간 줄이는 법") + 시연 참가 안내 (홍보물 ②)

### 카카오톡 오픈채팅 / 단톡방
- "품질관리", "IATF 16949", "FMEA", "APQP" 키워드 오픈채팅방
- 짧은 카드뉴스 + 신청 링크 형태(홍보물 ④)가 적합

### 기타
- **블라인드** (자동차·제조 라운지): 익명 실무자 풀
- **잡플래닛/사람인 기업 커뮤니티**: 제한적 보조 채널

> **주의**: 커뮤니티는 노골적 광고에 거부감이 큼. 반드시 **정보 제공 → 자연스러운 참가 안내** 순서. 운영자 사전 협의 권장.

## ★★ 2순위 — LinkedIn (B2B SNS)

QA/QC 매니저, 품질팀장, 생산기술 임원을 **직함 기반으로 정확히** 타겟팅.

- **타겟 검색 키워드**: 품질보증, 품질관리, QA, QC, 생산기술, IATF, FMEA + 회사규모(부품사)
- **운영 방법**:
  1. 회사 페이지 개설 → 데모 영상·고객가치 콘텐츠 정기 게시
  2. 타겟 담당자에게 개인화된 1:1 메시지(홍보물 ③)
  3. 자동차 부품/품질 관련 그룹에 사례 콘텐츠 공유
- **유료 광고**: LinkedIn은 직무·업종 타겟 정밀도가 높아 B2B 리드 광고에 적합 (예산 여유 시)

## ★ 3순위 — 콘텐츠 채널 (유튜브 / 블로그)

즉시 모집보다는 **신뢰 구축·인바운드 자산** 목적. 1·2순위를 뒷받침.

- **유튜브**: "도면 PDF 업로드 → 표준류 4종 자동생성" **실제 시연 영상** (1~3분). 모든 홍보물에 링크로 첨부 → 전환율 상승
- **블로그/브런치/회사 홈페이지**: "FMEA 작성 노하우", "IATF 16949 표준류 자동화 사례" 등 SEO 콘텐츠
- **페이스북 품질경영 그룹**: 보조 도달 채널

## 권장 실행 순서 (4주 모집 스프린트 예시)

```
1주차  기존 인맥·레퍼런스·제휴처 직접 접촉 → 1호 시연사 확보
       데모 영상(1~3분) 제작, 1페이지 소개서 완성
2주차  KAICA·KAP·KSA 등 협회에 공문 + 세미나 공동개최 제안
       LinkedIn 회사페이지 개설·첫 콘텐츠 게시
3주차  네이버 카페·카카오 오픈채팅 정보성 글 + 참가 안내
       LinkedIn 타겟 담당자 1:1 메시지 발송
4주차  반응 집계 → 시연 일정 확정, 무료 PoC 진행
       참가사 사례를 다음 라운드 콘텐츠로 재활용
```

## 모집 KPI (예시)
- 1차 목표: **시연 참가 신청 15~20개사**, 무료 PoC 진행 **5~8개사**, 파일럿 전환 **1~2개사**
- 채널별 유입·신청·전환을 추적해 다음 라운드 예산 배분
