# IATF 16949:2016 — PFMEA 관련 조항 요약

## 핵심 요약
IATF 16949:2016은 ISO 9001:2015를 기반으로 자동차 산업 특정 요구사항을 추가한 품질경영시스템 표준이다.
PFMEA는 APQP(사전 제품 품질 계획)의 핵심 도구로 명시되며, 조항 8.3과 8.5에서 직접 요구된다.

---

## PFMEA 관련 규칙·기준

### 조항 8.3 — 제품 및 서비스 설계·개발

**8.3.2.1 설계·개발 계획 (자동차 산업 추가)**
- PFMEA를 포함한 리스크 분석 수행 필수
- APQP 프로세스의 일환으로 PFMEA 작성 요구
- 제품 개발 초기 단계부터 PFMEA 수행 (선행 PFMEA 권고)

**8.3.3.3 특별특성 (자동차 산업 추가)**
- 고객이 정의한 특별특성(CC/SC) 기호 사용 필수
- 특별특성 식별 후 PFMEA, Control Plan, 작업표준서에 동일 표기 의무
- 특별특성 관련 고장유형은 추가 관리 방법 필요

**8.3.4.3 프로토타입 프로그램 (자동차 산업 추가)**
- 프로토타입 단계에서도 PFMEA 리스크 분석 요구

### 조항 8.5 — 생산 및 서비스 제공

**8.5.1.1 제조 공정 관리 계획 (자동차 산업 추가)**
- Control Plan(관리 계획서)은 PFMEA와 연계 필수
- PFMEA의 검출 관리 방법이 Control Plan에 반영되어야 함
- 특별특성 관리 방법이 Control Plan에 명시 필요

### 조항 8.7 — 부적합 출력물 관리

**8.7.1.4 고객 통보 (자동차 산업 추가)**
- PFMEA의 검출 관리 방법이 실제 공정에 구현·작동 확인 필요

### 조항 10.2 — 부적합 및 시정 조치

**10.2.3 문제 해결 (자동차 산업 추가)**
- 클레임·불량 발생 시 PFMEA 업데이트 의무
- 시정 조치 완료 후 PFMEA 재평가 및 AP 갱신

---

## PFMEA와 APQP 연계

APQP 5단계별 PFMEA 수행 시점:

| APQP 단계 | FMEA 활동 | 완료 기준 |
|---|---|---|
| 1단계: 프로그램 계획·정의 | FMEA 범위 및 팀 구성 | 팀 구성 완료 |
| 2단계: 제품 설계·개발 | 설계 FMEA 초안 | DFMEA 초안 완료 |
| 3단계: 공정 설계·개발 | PFMEA 초안 작성 | PFMEA 초안 완료 |
| 4단계: 제품·공정 유효성 확인 | PFMEA 최종 완성 및 CP 연계 | PPAP 제출 전 |
| 5단계: 피드백·평가·시정 | PFMEA 개정 및 갱신 | 변경 발생 시 |

## IATF 16949 내부 심사 — PFMEA 체크포인트

심사원이 PFMEA에 대해 확인하는 주요 항목:

| 심사 항목 | 합격 기준 |
|---|---|
| PFMEA가 실제 현행 공정을 반영하는가? | 최근 공정 변경 사항이 PFMEA에 반영됨 |
| 공정 변경 시 PFMEA 즉시 업데이트 체계 | 4M 변경 관리 절차에 PFMEA 업데이트 포함 |
| PFMEA ↔ Control Plan 특별특성 일치 | CC/SC 항목 번호·기호 동일 |
| AP=H 항목의 권고 조치 이행 증거 | 책임자·완료일·조치 내용 기입 |
| PFMEA 보관 기간 준수 | 양산 종료 후 최소 15년 |
| 작성·검토·승인자 서명 | 결재 이력 있음 |

## 주요 용어·정의

| 용어 | 정의 |
|---|---|
| APQP | Advanced Product Quality Planning — 사전 제품 품질 계획 |
| PPAP | Production Part Approval Process — 양산 부품 승인 프로세스 |
| Control Plan (CP) | 공정 특성 및 제품 특성을 관리하는 계획서 |
| 특별특성 | 제품 안전, 정부 규정 준수, 기능, 조립성에 영향을 미치는 특성 |
| 4M 변경 | Man/Machine/Material/Method — 변경 발생 시 PFMEA 업데이트 트리거 |

- [ ] 공정 변경(4M) 절차에 PFMEA 업데이트 요건이 포함되어 있는가?

- [ ] 불량 재발·클레임 발생 시 PFMEA 자동 개정 프로세스가 있는가?
